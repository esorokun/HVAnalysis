### Setup
Check out the code:
```
$ ssh://git@gitlab.cern.ch:7999/ligerlac/HVAnalysis.git
$ cd HVAnalysis/
```
Create a virtual environment, activate it and install dependencies
```
$ python -m venv venv/
$ . venv/bin/activate
$ pip install -r requirements.txt
```
